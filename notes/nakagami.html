Nakagami distribution

Parameters

  • \(\nu\): shape
  • \(a\): location
  • \(s\): scale
In the wikipedia article Nakagami distribution, the parameter \(m\) is the same as SciPy's \(\nu\). The SciPy scale parameter is \(s = \sqrt{\Omega}\). The parametrization in the wikipedia article does not include a location parameter.

PDF: \[ f(x, \nu, a, s) = \frac{2 \nu^{\nu}}{s\Gamma(\nu)}\left(\frac{x - a}{s}\right)^{2\nu - 1}\exp\left(-\nu\left(\frac{x - a}{s}\right)^2\right) \]

Logarithm of the PDF: \[ \log f(x, \nu, a, s) = \log 2 + \nu \log \nu - \log \Gamma(\nu) - \log s + \left(2\nu - 1\right)\log\left(\frac{x - a}{s}\right) - \nu \left(\frac{x - a}{s}\right)^2 \]

CDF: \[ F(x, \nu, a, s) = \frac{\gamma\left(\nu, \nu\left(\frac{x - a}{s}\right)\right)}{\Gamma(\nu)} \] where \(\gamma(u, v)\) is the lower incomplete Gamma function.

Mean: \[ E(X(\nu, a, s)) = a + s\frac{\Gamma(\nu + 1/2)}{m^{1/2}\Gamma(\nu)} \]

Maximum Likelihood Estimation

Likelihood function: \begin{equation} L(\textbf{x}, \nu, a, s) = \left(\frac{2 \nu^{\nu}}{s\Gamma(\nu)}\right)^N \prod_{i=1}^N \left(\frac{x_i - a}{s}\right)^{2\nu - 1}\exp\left(-\nu\left(\frac{x_i - a}{s}\right)^2\right) \label{eqn:likelihood} \end{equation}

Log-likelihood function: \begin{split} \ell(\textbf{x}, \nu, a, s) & = N\log 2 + N \nu \log \nu - N\log \Gamma(\nu) - N\log s + \left(2\nu - 1\right)\sum_{i=1}^{N}\log\left(\frac{x_i - a}{s}\right) - \nu \sum_{i=1}^{N}\left(\frac{x_i - a}{s}\right)^2 \\ & = N \left\{\log 2 + \nu\log\nu - \log\Gamma(\nu) - 2\nu\log s\right\} + (2\nu - 1)\sum_{i=1}^N\log(x_i - a) - \frac{\nu}{s^2} \sum_{i=1}^N (x_i - a)^2 \end{split}

Derivatives of the log-likelihood function: \begin{split} \frac{\partial \ell}{\partial \nu} & = N\left\{1 + \log\nu - \psi(\nu) - 2\log s \right\} + 2\sum_{i=1}^N \log(x_i - a) - \frac{1}{s^2}\sum_{i=1}^N (x_i - a)^2 \\ & = N\left\{1 + \log\nu - \psi(\nu)\right\} + 2\sum_{i=1}^N\log z_i - \sum_{i=1}^N z_i^2 \end{split} where \(z_i = (x_i - a)/s\) and \(\psi(\nu)\) is the digamma function, \(\psi(\nu) = \Gamma'(\nu)/\Gamma(\nu)\). \[ \frac{\partial \ell}{\partial a} = -(2\nu - 1)\sum_{i=1}^N \frac{1}{x_i - a} + \frac{2\nu}{s^2}\sum_{i=1}^N (x_i - a) \] \begin{split} \frac{\partial \ell}{\partial s} & = -\frac{2N\nu}{s} + \frac{2\nu}{s^3}\sum_{i=1}^N (x_i - a)^2 \\ & = \frac{2\nu}{s}\left(-N + \frac{1}{s^2}\sum_{i=1}^N (x_i - a)^2 \right) \end{split}

Location parameter fixed

If the location parameter \(a\) is fixed, then the scale determined by \(\frac{\partial \ell}{\partial s} = 0\) can be solved for explicitly: \[ s = \left(\frac{1}{N} \sum_{i=1}^N(x_i - a)^2\right)^{\frac{1}{2}} \] Then \(\frac{\partial \ell}{\partial \nu} = 0\) gives \[ \log\nu - \psi(\nu) = -1 + \frac{1}{N}\sum_{i=1}^N z_i^2 - \frac{2}{N}\sum_{i=1}^N \log(x_i - a) \] and \(\nu\) can be found by numerically inverting \(\log\nu - \psi(\nu)\).

Location parameter not fixed

Suppose all three parameters are available for fitting the distribution. Then the problem is not well posed. To see this, consider \(\eqref{eqn:likelihood}\), and without loss of generality, let \(x_1\) be the minimum of the elements of \(\textbf{x}\). Let \(\nu\) be any value in the interval \(0 \lt \nu \lt 1/2\), and let \(s\) be any positive value. We can make the likelihood \(L(\textbf{x}, \nu, a, s)\) as large as we like by letting \(a\) approach \(x_1\) from below, because in that case, the \(2\nu - 1\) is negative, and the term \(((x_1 - a)/s)^{2\nu-1}\) goes to infinity while the other terms in the expression approach constants.
