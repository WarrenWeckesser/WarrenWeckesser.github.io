xlog1py¶

mpsci.fun.xlog1py(x, y)¶

\(x\textrm{log}(1+y)\)

If x is 0 and y is not nan, 0 is returned.

This function is mathematically equivalent to mpsci.fun.xlogy(1 + y). It avoids the loss of precision that can result if y is very small.

See also: xlogy()
