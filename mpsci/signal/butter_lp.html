butter_lp¶

mpsci.signal.butter_lp(n, Wn)¶

Lowpass Butterworth digital filter design.

This computes the same result as scipy.signal.butter(n, Wn, output=’zpk’), but it uses mpmath, and the results are returned in lists instead of numpy arrays.
